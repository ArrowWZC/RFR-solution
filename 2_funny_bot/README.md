# gen_jokes
inspired from https://github.com/leido/char-rnn-cn

## Requirement
tensorflow >= 0.12.1
python3

## Train
`python3 gen_jokes.py 0`

## Generate jokes
`python3 gen_jokes.py 1`

Run command `tensorboard --logdir=logs` in the current directory, then get embedding visualization：
